# My-Python-Learning-Journey
Python 学习之旅的仓库，用于记录学习过程中的练习项目、代码和心得。

## 项目介绍
本仓库目前包含 1 个核心项目：猜数字游戏。
- 功能：系统随机生成 1-100 之间的整数，用户通过输入数字猜测，系统给出“大了”“小了”的提示，直到猜对并显示猜测次数。
- 技术点：Python 基础语法（循环、条件判断、随机数生成）。

## 运行方法
1. 环境要求：Python 3.6 及以上版本。
2. 运行步骤：
   - 克隆本仓库到本地：`git clone https://github.com/你的用户名/My-Python-Learning-Journey.git`
   - 进入项目文件夹：`cd My-Python-Learning-Journey`
   - 运行代码：`python 猜数字.py`（Windows 用 python，Mac/Linux 可能用 python3）

## 代码说明
核心代码逻辑：
1. 导入 random 模块，生成 1-100 的随机数作为谜底；
2. 初始化猜测次数为 0，通过 while 循环持续接收用户输入；
3. 判断用户输入的数字与谜底的大小关系，给出对应提示；
4. 猜对时，打印猜测次数并退出循环。

完整代码见：[猜数字.py](https://github.com/songningfu/My-Python-Learning-Journey/blob/main/%E7%8C%9C%E6%95%B0%E5%AD%97.py)（替换成你代码的实际链接，在 GitHub 上打开代码文件，复制浏览器地址即可）。
